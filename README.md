# potato-vue

#### 项目介绍
- 从前，有个土豆，他想去看世界，于是......


#### 软件架构
- [采用VUE Cli 3.0搭建](https://cli.vuejs.org/zh/guide/)


#### 安装教程
1. 
```
npm install
```

2. 
```
npm run serve
```

3. 
```
npm run build
```

4.本地访问地址
- Local:   http://localhost:8081/


#### 使用说明
- 登录页面 views-Login.vue
- 首页页面 views-Index.vue
